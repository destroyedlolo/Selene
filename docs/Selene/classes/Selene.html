Methods

Selene:SelWaitFor (...)
Wait for even to come or a task is scheduled. 

The process is put on hold and doesn't consume any processor resources until waked up.

Have a look on Selenites examples directory : this function is the most important one to achieve resources conservatives automation with Séléné. In addition, scheduled tasks are never launched if SelWaitFor() is not called.

Parameters:

  • ... list of SelTimer, SelEvent, file IO.
