Tables

SelTimer.create_arguments
Arguments for Create and Set

Fields:

  • when initial delay (seconds)
  • at initial launch (format HH.MM -> 10.45 for 10:45)
  • interval delay b/w next run (seconds)
  • clockid clock mode CLOCK_REALTIME (default) or CLOCK_MONOTONIC (see Linux documentation) [Create only]
  • ifunc function to run "immediately" when a timer is over
  • task function to put in task list
  • once avoid task duplication (don't push it again if already in the todo list)
