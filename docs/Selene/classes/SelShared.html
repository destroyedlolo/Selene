Class SelShared

Shared variables : share a value b/w threads.

Each threaded task run within dedicated state : consequently, they can't share Lua's objects with others. Shared stuff are special classes to share something among threads.

Have a look on the source code for C interface.

SelShared:TaskOnceConst (once)
Transcode "ONCE" code.

ONCE : don't push if the task is already present in the list. MULTIPLE : task will be pushed even if already present. LAST : if already in the list, remove and push it as last entry of the list.

Parameters:

  • once string

Returns:

    code
SelShared:PushTask (function, once)
Push a task to the waiting list

ONCE : don't push if the task is already present in the list. MULTIPLE : task will be pushed even if already present. LAST : if already in the list, remove and push it as last entry of the list.

Parameters:

  • function function
  • once true : ONCE (default), false : MULTIPLE or SelShared.TaskOnceConst("LAST")
